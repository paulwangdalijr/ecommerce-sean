This is a demo app developed by Paul Wangdali Jr.

Angular 6, Node.js, Express.js, SQLite, Bootstrap 4 using Bootswatch Lux Theme and utilizing some of the NPM packages to rapidly develop this amazing E-commerce website :)

Some of the functionalities are yet to be implemented!!!


Modules:

Admin Module - Admin users can add/edit/delete products, add/delete categories and view orders - Orders are generated on user's checkout

User Module - A user can view products, add products to cart, perform checkout and place an order.

Sign up Module - A user can create an account, creating an account will allow the user to save profile information which will help him to checkout faster as the checkout form will be auto populated based on his profile.

Sign in Module - A user can sign through his created account or through Facebook.

Cart Module - A user can add products, view and modify his cart.

Check out Module - Once a user has decided to confirm his purchase, he can proceed to check out and place an order. An order number will be generated.
